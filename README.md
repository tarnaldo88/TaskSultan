# TaskSultan

A modern SaaS task management app inspired by Jira. Built with ReactJS for the frontend. See PRD.md for full product requirements. More readme being added in future

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm start
   ```

## Project Structure
- `/src` - React source code
- `/public` - Static assets

## Features
- User auth (email/OAuth)
- Workspaces & projects
- Kanban, list, and calendar task views
- Comments, notifications, attachments
